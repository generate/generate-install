## Usage

Register as a plugin or sub-generator:

```js
// plugin
module.exports = function(app) {
  app.use(require('{%= name %}'));
};

// sub-generator
module.exports = function(app) {
  app.register('foo', require('{%= name %}'));
};
```

See the [generate docs](https://github.com/generate/generate) for more details.
